# gh-pico

gh-pico is a small GitHub Copilot coding agent CLI built by wiring the pi-mono agent framework to the Copilot SDK. The goal is simple: make a practical, hackable agent experience that feels native in the terminal while still benefiting from GitHub's model access.

## What we're doing

We're building a CLI that can run locally or connect to remote environments like Codespaces, SSH targets, and devcontainers. It brings together agent sessions, tools, transport layers, and Copilot-powered model access in one place.

## What we're trying to achieve

- Make agentic coding feel lightweight and reliable
- Keep the architecture easy to understand and extend
- Let the same workflow follow you from local projects to remote workspaces

It's early, focused, and meant to stay practical.
